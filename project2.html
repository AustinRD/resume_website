Semester Planner

  • Source Code

This project was requested by the Software Engineering department at SUNY New Paltz. The goal was to create a tool that could be used by Academic Departments at colleges to automatically schedule a given semester's courses within some date range without conflicts given some set of restrictions. This project is a work in progress, and was incomplete was of December 2020.

No external libraries or frameworks were used per request. The technologies used were HTML5, CSS, JavaScript, PHP and a MySQL relational-database. The features of this version are as follows:

  • User types: Scheduler (Admin/Department Chair) and Viewer (Instructors)
  • Add/Remove Courses (Credit-Hours Required, Pre-requisite courses)
  • Allow Instructors to add Availability.
